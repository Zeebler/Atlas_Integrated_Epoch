# Changelog

## 1.0.0

- Initial production-ready release.
- Integrates Atlas dungeon maps into the Blizzard world map experience.
- Adds dungeon selector, boss list, boss loot panel, and optional AtlasLoot loot linking.
- Adds support for AtlasLoot trash loot as a final unnumbered `Trash` entry.
- Adds ElvUI-aware styling and Blizzard fallback styling.
- Adds compatibility safeguards for pfQuest, ElvUI, Magnify, Atlas, and AtlasLoot.
- Adds Interface Options settings and slash commands.
